# Nautilus Script

My personal Nautilus scripts.

## Install

Via SCons

``` bash
$ scons -Q install
```

Custom location can be specified by option `--install-dir`.

## Uninstall

``` bash
$ scons -Qc install
```

If installed to a custom location, add option `--install-dir`.

## Usage

Right click any file/directory in GNOME Files.
